<h2 align="center"> Relação de comandos Kubernetes. </h2>

<p align="center"> 🛠️ Este capítulo apresenta um lista de comandos do Kubernetes no formato 'Cheat Sheet'. </p>

| Comando | Descrição |
|---------|----------|
| kubectl cluster-info | Apresenta informações do cluster Kubernetes. |
| kubectl cluster-info **dump** | Apresenta as informações em formato longo e mais detalhados. |
| kubectl get nodes **-o wide** | Lista todos os nodes no cluster e apresenta mais informações, como IP e sistema operacional. |
| kubectl config view | Para visualizar a configuração do seu cluster. |
| kubectl get pods | Lista todos os pods no atual namespace. |
| kubectl get pods **-o wide** | Mostra informações detalhadas sobre um Pod. |
| kubectl get pods **-l label** | Ver os Pods com alguma label específica. |
| kubectl get pod **name** | Ver um Pod específico. |
| kubectl describe **pod** | Ver detalhes de um Pod. |
| kubectl logs **pod** | Ver os logs de um Pod. |
| kubectl exec -it **pod** | Executar um comando em um Pod. |
| kubectl delete pod **pod** | Deletar um pod. |
| kubectl explain **resource** | Ver a documentação de um recurso. |
