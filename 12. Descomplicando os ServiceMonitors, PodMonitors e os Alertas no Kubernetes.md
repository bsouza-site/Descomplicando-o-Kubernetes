<h3 align="center"> 12.1. O que é um ServiceMonitor? </h3>

Um dos principais recursos que o Kube-Prometheus utiliza é o ServiceMonitor. O ServiceMonitor é um recurso do Prometheus Operator que permite que você configure o Prometheus para monitorar um serviço. Para isso, você precisa criar um ServiceMonitor para cada serviço que você deseja monitorar.

O Kube-Prometheus já vem com vários ServiceMonitors configurados, como por exemplo o ServiceMonitor do API Server, do Node Exporter, do Blackbox Exporter, etc.

Utilizando o **kubectl**, veja alguns comandos de administração:
* **Ver os ServiceMonitor criados**: kubectl get servicemonitors -n monitoring  
* **Ver detalhes de um ServiceMonitor**: kubectl get servicemonitor prometheus-k8s -n monitoring -o yaml

Neste momento, será criado o Deployment e Service no Kubernetes. Antes disso:  
* **Verifique se o cluster está de pé**: kubectl get nodes  
* **Ver se o Kube-Prometheus está instalado**: kubectl get customresourcedefinitions.apiextensions.k8s.io  
* **Para ver todos os ServiceMonitors criados**: kubectl get servicemonitors -A
